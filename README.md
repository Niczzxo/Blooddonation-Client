# 🩸 Blood Donation Application

A complete **MERN Stack Blood Donation Platform** that connects blood donors with people in urgent need. The goal is simple: **save lives by making blood donation faster, easier, and more reliable**.

## ✨ Key Features

### 👤 Authentication & Roles

* Secure Registration & Login (Email + Password)
* Firebase Authentication
* Role-based Access Control

  * 🧑 Donor
  * 🧑‍🤝‍🧑 Volunteer
  * 👑 admin

---

### 🩸 Blood Donation System

* Create Blood Donation Requests
* Edit & Delete Own Requests
* Public Blood Requests Page (Only **Pending** requests)
* Private Request Details Page
* **Donate Now** Modal with Confirmation
* Donation Status Flow:

  * `pending` → `inprogress` → `done`

---

### 📊 User Dashboard

* My Donation Requests
* Pagination Support
* Filter by Status (Pending / In Progress / Done)
* Clean & User-friendly Dashboard UI

---

### 👑 admin Panel

* View All Users
* Block / Unblock Users
* Change User Roles (Donor ↔ Volunteer ↔ admin)
* Secure admin Routes

---

### 💳 Funding & Payments

* Stripe Payment Integration
* Secure Online Donations

---

### 🎨 UI & UX

* Fully Responsive (Mobile + Desktop)
* Tailwind CSS + DaisyUI
* Lottie Animations for Loaders
* Toast Notifications for Actions
* Clean, Modern & Accessible Design
